## Google Cloud Functions

### Starter Kit

Just clone the repo and start developing.

### Development

```bash
# 1st Terminal
npm run gcp-dev

# 2nd Terminal
npm run dev
```

### Deployment

```
# gcloud cli has to be installed and you need to be logged in. See links below.

npm run deploy
```
